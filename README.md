## 🌟 Features

### 🔥 Live Markdown Preview  
- Edit Markdown on the left  
- See real-time HTML output on the right  
- Mobile-friendly tab mode  

### 🌗 Dark & Light Theme  
- Toggle anytime  
- Saves your preference  

### 🔄 Sync Scrolling  
- Editor & preview scroll together  
- Can be disabled  

### 🔢 Line Numbers  
- Auto-generated  
- Always up to date  

### ✍️ Live Word & Character Counter  
- Accurate  
- Updates instantly  

### 💾 Autosave  
- Saves content locally  
- Survives page refresh  

### 📤 Export to HTML  
- One-click clean HTML export  

### 📱 Fully Responsive  
- Optimized for phone, tablet, and desktop  

### ↔️ Draggable Split View  
- Adjust editor & preview size freely

## 📘 How to Use

1. Open `index.html`
2. Type Markdown
3. Preview updates instantly
4. Toolbar:
   - 🔄 Sync  
   - 🌗 Theme  
   - 📤 Export  
5. Auto-save keeps your content safe
